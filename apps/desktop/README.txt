FTC Sim — Quick start (friends)
================================

1. Extract the zip anywhere on your PC.
2. Run FTC Sim.exe (Windows may show SmartScreen — click "More info" → "Run anyway").
3. Choose:
   - Play Solo — practice alone
   - Host Match — you run the server; share the LAN address shown
   - Join Match — enter the host's address (e.g. 192.168.1.50:5191)

Multiplayer tips
----------------
- Host and friends must be on the same Wi‑Fi/LAN, OR the host uses a free playit.gg TCP tunnel to port 5191.
- After Host opens the sim, press INF in the toolbar to drive.
- Friends joining only need this zip — no Node.js or dev tools.

Internet play (host only)
-------------------------
1. Install https://playit.gg
2. Create a TCP tunnel → 127.0.0.1:5191
3. Share the playit address (e.g. xyz.playit.gg:54321) with friends
